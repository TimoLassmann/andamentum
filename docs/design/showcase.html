Andamentum · v1.0

A design system that reads like a book.

Andamentum is a scholarly, letterpress-honest design system for documents and the apps that produce them. Cream paper, serif headings, hairline rules, muted earth tones — extended to a complete vocabulary of app chrome.

Philosophy Restrained, considered, typographic. The page is the foundation; chrome is a margin around it. Type Source Serif 4 for prose, Inter for UI and metadata, SF Mono for code. Palette Warm cream paper, ink in seven steps, muted semantic tones. Dark mode is vellum-by-lamplight, not negative space. Coverage Documents, app chrome, lists, chat, instrumentation, settings, tagging — everything a productivity or agent app needs.
